# 🚀 YouTube Creator OS Dashboard

A premium, glassmorphic analytics and content strategy dashboard designed for high-growth YouTube creators. This "Creator OS" helps you track performance, manage multiple channels, brainstorm viral ideas, and monitor competitor intelligence in one sleek interface.

Dashboard preview image intentionally omitted in this repo copy.

## ✨ Key Features

### 📊 Advanced Analytics Suite
- **Engagement Score**: Real-time calculation of audience interaction quality.
- **Best Time to Upload**: Data-driven analysis of your most successful upload windows.
- **Subscriber Growth**: Historical logging and multi-channel comparison charts (via Chart.js).
- **Milestone Tracking**: Automatic calculation of your next 1,000 subscriber milestone.

### 🛡️ Competitor Intelligence (Interactive Sandbox)
- **100+ Viral Hooks**: Pre-loaded intelligence for 20 competitors across "Real-life Stories" and "Pet Adventures".
- **Click-to-Watch**: Instantly visit viral videos or trigger smart YouTube search results for any hook.
- **Inspiration Bank**: High-performance benchmarks from industry leaders like *The Dodo*, *Tucker Budzyn*, and *Delafé Testimonies*.

### 💡 Creator Workflow Tools
- **Idea Bank**: A dedicated brainstorming sandbox to capture and promote viral concepts.
- **Production Checklists**: Track SEO, Thumbnail, and Description readiness for every planned upload.
- **Multi-Channel Management**: Seamlessly toggle between distinct channel strategies (e.g., L-P-B-Z and Tozz Jerry).

### ⚙️ Management & Portability
- **YouTube API Integration**: One-click sync to pull live subscriber counts and video metadata.
- **Data Portability**: Full JSON Export/Import capabilities for backup and data ownership.
- **Glassmorphic UI**: A state-of-the-art interface with dark/light mode, smooth animations, and premium aesthetics.

## 🚀 Getting Started

1.  **Clone the Repo**:
    ```bash
    git clone https://github.com/batyaboyo/yt_tracker.git
    ```
2.  **Open locally**:
    Simply open `index.html` in your browser.
3.  **Setup API**:
    Navigate to the **Settings** tab and add at least one YouTube Data API v3 key to enable live syncing.

## 🛠️ Tech Stack
- **Frontend**: Vanilla HTML5, Vanilla CSS3 (Custom Properties, Glassmorphism).
- **Logic**: Vanilla JavaScript (ES6+), YouTube Data API v3.
- **Charts**: [Chart.js](https://www.chartjs.org/) for data visualization.
- **Storage**: Browser LocalStorage for persistent, private data.
